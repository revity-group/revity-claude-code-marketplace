# Revity Claude Code Marketplace

A collection of Claude Code plugins for full-stack development workflows.

## What is this?

This marketplace provides sharable agentic workflows that are specific to Revity tailored to different use cases and clients and internal use

## Installation

1. Open Claude Code and run the `/plugins` command
2. Select **Add marketplace**
3. Enter the repository URL:
   ```
   https://github.com/revity-group/revity-claude-code-marketplace
   ```
4. Browse and install available plugins

After installation, restart Claude Code to load the new commands.
